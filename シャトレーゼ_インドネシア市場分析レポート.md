# シャトレーゼ インドネシア市場分析レポート

**作成日**: 2026年2月5日
**分析フレームワーク**: 市場 → プレイヤー（競合/自社） → 顧客

---

## エグゼクティブサマリー

インドネシアの菓子・ベーカリー市場は、2億7,000万人の人口と拡大する中間層を背景に成長を続けている。シャトレーゼは2017年の参入以来、Gobelグループとの合弁で事業を展開し、2025年度末までに450店舗体制を目指す積極的な拡大計画を進行中。競合にはローカル老舗（Holland Bakery）、プレミアム層（The Harvest）、韓国系（Tous les Jours、Paris Baguette）が存在し、各社SNSを活用したマーケティングを展開している。

## 1. 市場分析

### 1-1. カテゴリ市場の傾向

**判定: 拡大傾向**

| 指標 | 数値 |
|------|------|
| 菓子市場規模（2024年） | USD 23.8億（約3,600億円） |
| 予測市場規模（2033年） | USD 32.7億 |
| 菓子市場CAGR | 3.25%（2025-2033） |
| ベーカリー市場規模（2025年） | USD 547億 |
| ベーカリーCAGR | 5.70%（2025-2030） |

インドネシアはASEANベーカリー市場の**27.89%**を占める最大市場。

### 1-2. カテゴリ市場のボリューム

| 指標 | 数値 |
|------|------|
| 人口 | 約2億7,000万人（ASEAN最大） |
| 中間層比率 | 人口の約66%が中間層およびその予備軍 |
| 消費支出シェア | 中間層が総家計支出の81.49%を占める |
| チョコレート菓子シェア | 市場の50%以上 |
| 砂糖菓子シェア | 市場の38% |

### 1-3. カテゴリ市場傾向の要因

#### 成長ドライバー

- 中間層の拡大と可処分所得の増加
- 急速な都市化（便利な食品への需要増）
- 西洋化された食習慣の浸透
- SNS・韓流ドラマによるカフェ文化・スイーツブームの拡散
- ギフト文化（ラマダン、クリスマス、旧正月）

#### 消費者トレンドの変化

- ヘルスコンシャス層の増加（低糖・グルテンフリー需要）
- プレミアム・グルメ志向の高まり
- オンライン購買の急成長（CAGR 11.49%）
- コンビニエンス重視（シングルサーブ、ポーションコントロール）

#### 注意すべきリスク要因

- 経済不安による消費者の節約志向（41%が支出に慎重）
- インフレ圧力による原材料コスト上昇
- 米国貿易政策による輸入物価上昇リスク

## 5. SWOT分析（シャトレーゼ）

### Strengths（強み）

- 「日本直輸入」「日本品質」のブランド力
- 甘さ控えめで健康志向層に訴求
- Gobelグループとの強力なパートナーシップ
- 現地工場による価格競争力（Rp 15,000以下の商品展開可能）
- カカオ農家支援によるサステナビリティ訴求

### Weaknesses（弱み）

- 現時点での店舗数は競合に劣る（約40店舗 vs Holland Bakery 470店舗）
- ハラル認証の取得状況が不明確
- SNSフォロワー数で韓国系競合に劣る

### Opportunities（機会）

- 2億7,000万人市場へのアクセス拡大余地
- 新工場稼働（2026年）による価格競争力強化
- 「日本品質×手頃な価格」のブルーオーシャン
- 健康志向トレンドとの親和性
- オンライン・デリバリーチャネルの成長
- ラマダン等のギフトシーズン需要

### Threats（脅威）

- 韓国系ベーカリー（Paris Baguette、Tous les Jours）の積極展開
- 消費者の節約志向の高まり
- Holland Bakeryの圧倒的店舗網
- ハラル認証の重要性増大
- 原材料コスト上昇リスク

## 6. 戦略提言

### 6-1. ポジショニング戦略

### 6-2. SNSマーケティング戦略

| 施策 | 詳細 | 参考競合 |

#### 短期（〜6ヶ月）
